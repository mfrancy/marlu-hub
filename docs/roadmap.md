# Roadmap

## MVP 1 - Catálogo e Produtos

Objetivo: disponibilizar um catálogo online e permitir o controle básico de produtos.

### Produtos

* [ ] Cadastrar produto
* [ ] Editar produto
* [ ] Excluir produto
* [ ] Inativar produto
* [ ] Listar produtos
* [ ] Buscar produtos

### Catálogo

* [ ] Exibir produtos disponíveis
* [ ] Filtrar por categoria
* [ ] Visualizar detalhes do produto
* [ ] Botão de contato via WhatsApp

### Infraestrutura

* [ ] Criar projeto Angular
* [ ] Configurar PrimeNG
* [ ] Configurar Bootstrap
* [ ] Estruturar pastas do projeto
